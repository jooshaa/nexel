# Design
- For hero sections: Use 40/60 split (text/image), NOT 50/50. Confidence: 0.85
- Hero images should scale 1.2-1.4x and overflow container for premium effect. Confidence: 0.85
- Prefer Dyson/Apple-style premium hero layouts with dominant images. Confidence: 0.80
- For pagination bullets: Use thin line style (h-[2px]), NOT thick bars. Confidence: 0.70
- For product grids: Use floating items WITHOUT card backgrounds (NO bg-white, bg-gray, bg-black). Confidence: 0.85
- Product items should be minimal with only image + title + subtitle, NO card containers. Confidence: 0.85
- Prefer Apple/Xiaomi-style clean product layouts with transparent/floating appearance. Confidence: 0.80
- For hover effects: Use subtle scale (1.05-1.1) + translateY(-4px) only, NO shadows. Confidence: 0.80
